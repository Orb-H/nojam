주어진 수열에서 0이 나오면 이전 원소를 지우는 규칙이 있을 때, 합을 구하는 문제입니다. 스택을 만들고, 입력받는 수가 0이면 pop, 아니면 push를 한 후, 마지막에 원소를 모두 꺼내서 합을 구한 후 출력하면 됩니다.

이 문제를 푼 시점 기준으로 백준의 [단계별로 풀어보기](http://noj.am/p/s) 20단계 **스택**의 두 번째 문제입니다.
