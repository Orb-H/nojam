규칙을 찾아 정수 N에 대한 모양을 출력하는 문제입니다. 규칙은 2\*N-1부터 1까지 길이를 2씩 줄여가면서 `*`을 출력하다가 1부터는 다시 2\*N-1까지 2씩 늘려가면서 `*`를 출력하는 것입니다. 또한 각 줄마다 중심이 일치하게끔 적당한 양의 공백도 같이 출력해야합니다.
