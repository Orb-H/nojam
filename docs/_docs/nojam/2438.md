자연수 N에 대해 첫 번째 줄에는 별 하나, 두 번째 줄에는 별 두개, ... , N번째 줄에는 별 N개를 출력하는 문제입니다. 반복문을 이용하여 루프 변수를 1부터 N까지 바꿔가며 각 루프마다 `*`를 루프 변수만큼 출력하면 됩니다.

이 문제를 푼 시점 기준으로 백준의 [단계별로 풀어보기](http://noj.am/p/s) 3단계 **for문**의 아홉 번째 문제입니다.
