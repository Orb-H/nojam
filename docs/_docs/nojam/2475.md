5개의 정수의 제곱의 합을 10으로 나눈 나머지를 구하는 문제입니다. 합을 계산할 변수를 준비한 후, 각 입력받은 정수마다 제곱(`x*x`)한 값을 합 변수에 더해줍니다. 마지막으로 이 변수를 10으로 나눈 나머지를 출력하면 됩니다.
