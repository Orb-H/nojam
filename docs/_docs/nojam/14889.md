각 사람쌍마다 시너지 값이 주어져있을 때, 사람들을 두 팀으로 나누었을 때 각 팀의 시너지 합의 차이의 최소값을 구하는 문제입니다. 재귀 함수를 이용해 사람을 스타트 팀 또는 링크 팀에 배치하는 경우를 만들고 각 경우에 대한 시너지 합의 차이의 최소값 중 작은 값을 반환하게 하면 됩니다.
