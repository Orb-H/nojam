연도가 윤년인지 판단하는 문제입니다. n의 배수인지 확인하는 것은 나머지 연산자를 통해 할 수 있습니다. n으로 나눈 나머지가 0이면 해당 숫자는 n의 배수인 것이죠. 나머지 연산과 논리 연산자를 이용하여 입력받은 연도가 4의 배수이며 100의 배수가 아니거나, 400의 배수인지 체크할 수 있습니다.
