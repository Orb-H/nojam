자연수 N에 대해 구구단의 N단을 출력하는 문제입니다. 반복문을 이용하여 루프 변수를 1부터 9까지 바꿔가며 구구단을 한 줄씩 출력하면 됩니다.

반복문을 사용해야 하는 문제이지만 일반적인 언어의 경우 반복문보다 하드 코딩이 성능이 더 좋다고 합니다.
