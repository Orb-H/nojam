사이트와 비밀번호의 집합과 비밀번호를 알고 싶은 사이트 목록이 주어졌을 때 알맞은 비밀번호를 출력하는 문제입니다. 사이트 주소가 key이고 비밀번호가 value인 map 구조에 사이트와 비밀번호를 저장하고, 이후 주어지는 사이트 목록에 대해 map에 사이트를 입력하여 올바른 비밀번호를 받아 출력하면 됩니다.
