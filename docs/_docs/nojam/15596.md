길이가 n인 정수 배열이 매개 변수로 주어졌을 때 배열의 모든 원소의 합을 반환하는 함수를 작성하는 문제입니다. 많이 하던 배열의 전체 합 구하기를 하나의 함수로 정의하는 것이 목표입니다.

이 문제를 푼 시점 기준으로 백준의 [단계별로 풀어보기](http://noj.am/p/s) 7단계 **함수**의 첫 번째 문제입니다.
