두 수의 최대공약수와 최대공배수를 구하는 문제입니다. 최대공약수를 구하는 것은 유클리드 호제법을 이용하면 됩니다. 그리고 최대공약수와 최소공배수의 관계인 $$G\times L=A\times B$$에 앞에서 구한 최대공약수를 대입하면 최소공배수도 쉽게 구할 수 있습니다.
