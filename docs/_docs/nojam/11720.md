길이 N의 `0~9`로 이루어진 문자열의 각 숫자의 합을 출력하는 문제입니다. 문자는 정수의 형태로 저장되고 그 정수는 아스키 코드라는 점을 생각하면 쉽게 풀 수 있을 것입니다. 문자열의 문자를 순회하면서 **문자의 합이 아닌** 아스키 코드로부터 `0~9`에 해당하는 정수를 유추한 후 이것을 합산해야 합니다.
