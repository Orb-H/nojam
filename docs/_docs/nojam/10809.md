각 알파벳에 대하여 주어진 문자열에 처음 등장하는 위치를 찾는 문제입니다. 각 알파벳의 위치를 의미하는 크기가 26인 배열을 선언하고 -1로 초기화한 후, 주어진 문자열을 순회하면서 나오는 알파벳의 위치를 배열에 저장하고 이를 출력하면 됩니다.

이 문제를 푼 시점 기준으로 백준의 [단계별로 풀어보기](http://noj.am/p/s) 8단계 **문자열**의 세 번째 문제입니다.
