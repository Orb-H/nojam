두 수의 약수/배수 관계를 출력하는 문제입니다. 나머지 연산을 이용하여 구할 수 있는데, 첫 번째 수를 두 번째 수로 나눈 나머지가 0이면 `multiple`, 반대로 두 번째 수를 첫 번째 수로 나눈 나머지가 0이면 `factor`, 둘 다 아니라면 `neither`를 출력하면 됩니다.
